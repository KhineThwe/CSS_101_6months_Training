2. Border Style Dotted

3. Border Style Dashed

4. Border Style Solid

5. Border Style groove

Groove : It makes the border with a grooved line segment,which makes us feel that it is going inside.

6. Border Style inset

Inset : It makes the border with a embedded line segment,which makes us feel that it is fixed deeply on the screen.

7. Border Style outset

Outset : It is the opposite of inset. It makes the border with a line segment,which appears it to be coming out.

8. Border Style ridge

Ridge : It is the opposite of groove. It makes the border with a line segment,which makes us feel that it is coming out.

9. Border Style double

Double : double solid line

10. Border Style hidden

Hidden : It is used to make the top border invisible,like none,except in case of border conflict resolution of table elements.

11. Border Style initial

initial : It is used to set the default value of the element
