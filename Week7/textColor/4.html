HSL(0,100%,50%) represents red

  • The first value 0 represents hue and takes a numeric value ranging from 0 to 360. The value 0 represents red,120 represents green and 240 represents blue and also 360 represents red.
  • The second value 100% represents saturation which refers to the intensity of hue. The value ranges from 0%(fully gray) to 100%(fully saturated)
  • The third value represents lightness which refers to the brightness of the color. The value ranges from 0%(fully black) to 100%(fully white)

HSLA - A means Alpha - opacity

hsla(0,100%,50%,0.4)

The final value of hsla 0.4 represents the opacity whose value ranges from 0(fully transparent) to 1(fully opaque)
