RGB(255,0,0) represents red

RGB(100%,0,0) represents red

RGB(0,255,0) represents green

RGB(0,100%,0) represents red

RGB(0,0,255) represents green

RGB(0,0,100%) represents red

RGBA(255,0,0,0.5) represents red with opacity 0.5 if opacity value is 0(full transparent) and if opacity value is 1(fully opaque)
