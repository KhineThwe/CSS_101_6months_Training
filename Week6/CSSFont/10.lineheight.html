CSS Line-Height Property is used to set the height of the line box.

The default line height(also know as line spacing) for most web browsers is usually around 1.2 to 1.4 times the font-size, depending on the browser and the specific font being used.This means that there is a slight space between each line of text to make it more readable and visually appealing.

  • number: specifics the number that is multiplied with current font size to set the line height.
  • length: sets the line height in px,pt,cm,etc.....
  • % : sets the line height in percent of the current font size.
  • initial : sets the value to default value : i.e normal
  • inherit: inherits the property value from its parent element...

If we set the line-height property to "1.2" and font-size of 20 px.
Font Size : 20px
Line Height: 20px * 1.2 = 24 px
