CSS Font Properties are used to adjust the appearance of the text in an HTML document. Using the CSS fonts properties,we can customize the font family,size,weight,style, and color of text

Basic Font Properties:
1.font-family : sets size of the font
2.font-weight : sets the thickness i.e increase the boldness or lightness of the font
3.font-style : sets the font to italic or oblique
4.font-variant : changes the font to small-caps
5.font-stretch : expands or narrows the text
6.line-height: sets the distance between lines of the text
